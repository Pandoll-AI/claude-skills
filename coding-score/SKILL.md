# Coding Score - AI 코드 품질 평가 프레임워크

프로젝트의 코드 품질을 6가지 핵심 영역에서 평가하고 상세 리포트를 생성합니다.

## 평가 영역 (각 0-10점)

| 영역 | 설명 |
|------|------|
| **기능적 정확성** | 테스트 통과율, 커버리지, 에러 핸들링 |
| **성능 및 효율성** | 알고리즘 복잡도, 메모리 사용, 쿼리 효율성 |
| **보안 취약점** | 인젝션, 인증, 의존성 취약점 |
| **유지보수성** | 린트, 복잡도 메트릭, 코드 스타일 |
| **아키텍처** | DB 스키마 설계, API 구조, 모듈화 |
| **사용자 경험** | 에러 메시지, 접근성, 반응형 |

## 사용법

### 기본 평가 실행
```bash
python scripts/analyze.py /path/to/project
```

### 상세 리포트 생성
```bash
python scripts/analyze.py /path/to/project | python scripts/report.py
```

## 워크플로우

1. **프로젝트 언어 감지**: Python/JavaScript 자동 판별
2. **도구 기반 분석**: 설치된 도구(pytest, eslint, bandit 등) 활용
3. **정적 패턴 분석**: 도구 없이도 수행 가능한 패턴 탐지
4. **점수 산정**: 각 영역 0-10점으로 평가
5. **리포트 생성**: 상세 체크리스트 + 개선 권고

## 지원 도구

### Python 프로젝트
- `pytest` - 테스트 실행
- `coverage` - 커버리지 측정
- `pylint` / `flake8` - 린트
- `bandit` - 보안 검사
- `radon` - 복잡도 분석

### JavaScript/TypeScript 프로젝트
- `jest` / `mocha` - 테스트 실행
- `eslint` - 린트
- `npm audit` - 의존성 보안

도구가 설치되지 않은 경우, 정적 패턴 분석으로 대체됩니다.
